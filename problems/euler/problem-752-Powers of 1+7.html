Powers of 1+7

Problem 752

When (1+7) is raised to an integral power, n, we always get a number of the form (a+b7).
We write (1+7)n=α(n)+β(n)7.

For a given number x we define g(x) to be the smallest positive integer n such that: α(n)≡1(modx)and β(n)≡0(modx) and g(x)=0 if there is no such value of n. For example, g(3)=0, g(5)=12.

Further define G(N)=∑x=2Ng(x) You are given G(102)=28891 and G(103)=13131583.

Find G(106).
