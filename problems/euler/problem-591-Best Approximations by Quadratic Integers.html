Best Approximations by Quadratic Integers

Problem 591

Given a non-square integer d, any real x can be approximated arbitrarily close by quadratic integers a+bd, where a,b are integers. For example, the following inequalities approximate π with precision 10−13:
43756361915202−6188084046055<π<7211333155822−1019836515172
We call BQAd(x,n) the quadratic integer closest to x with the absolute values of a,b not exceeding n.
We also define the integral part of a quadratic integer as Id(a+bd)=a.

You are given that:

  • BQA2(π,10)=6−22
  • BQA5(π,100)=265−55
  • BQA7(π,106)=560323−2117817
  • I2(BQA2(π,1013))=−6188084046055

Find the sum of |Id(BQAd(π,1013))| for all non-square positive integers less than 100.
