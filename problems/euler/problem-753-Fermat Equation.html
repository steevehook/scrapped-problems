Fermat Equation

Problem 753

Fermat's Last Theorem states that no three positive integers a, b, c satisfy the equation an+bn=cn for any integer value of n greater than 2.

For this problem we are only considering the case n=3. For certain values of p, it is possible to solve the congruence equation: a3+b3≡c3(modp)

For a prime p, we define F(p) as the number of integer solutions to this equation for 1≤a,b,c<p.

You are given F(5)=12 and F(7)=0.

Find the sum of F(p) over all primes p less than 6000000.
